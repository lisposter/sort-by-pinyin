# sort-by-pinyin
[![NPM version](https://img.shields.io/npm/v/sort-by-pinyin.svg?style=flat)](https://www.npmjs.org/package/sort-by-pinyin)


------

## Installation

```bash
$ npm install sort-by-pinyin
```

## Usage
```js
var sort = require('sort-by-pinyin');

console.log(sort(['上海', '北京', '厦门', '杭州', '长沙']));
```
